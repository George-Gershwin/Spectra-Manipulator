User Namespace
==============


The methods located in module ``user_namespace.py`` are imported to the console, thus, they are accessible through console.
Some methods are often very useful (mainly functions :func:`add_to_list`,  :func:`set_xy_range` and  :func:`copy_to_clipboard`).
